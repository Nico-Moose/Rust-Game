# Core Systems

## 1. Resources
### Basic resources
- **Wood** — starter construction, primitive crafting, upkeep for wood structures.
- **Stone** — stronger base upgrades, early defensive progression.
- **Metal Ore** — refined into metal fragments or base metal.
- **Cloth** — bandages, clothing, primitive gear.
- **Food** — survival, healing support, buffs, expedition sustain.
- **Fuel** — generators, processing, some crafting systems.

### Progression currencies
- **Scrap** — the main research and progression currency.
- **WizeGold / stream-linked bonus currency** — limited auxiliary reward layer tied to Twitch/WizeBot systems.

### Components
- gears
- pipes
- springs
- wires
- circuit boards
- mechanisms
- optics
- military parts

### Rare resources
- high quality metal
- rare chemicals
- advanced electronics
- elite reactor parts

## 2. Crafting and tech progression

- primitive tools

- camp items
- basic medicine

### Workbench Tier 2
Unlocks:
- better tools

### Workbench Tier 3

### Research model
Players spend scrap to unlock recipes from tech branches:
- tools
- weapons
- building
- defense
- utilities
- medicine
- raid tech
- electricity

## 3. Base building
### Core building pieces
- square foundation
- triangle foundation
- wall
- doorway
- door
- ceiling/floor
- ladder hatch or stairs
- storage box
- workbench room
- airlock pieces

### Base core / authorization cupboard
Acts as the control node for the base.
- Defines ownership and allied access
- Stores upkeep resources
- Powers some advanced mechanics conceptually
- Prevents chaotic ownership disputes

### Upkeep
The base consumes resources over time. Larger, more advanced bases require more upkeep. This acts as an anti-bloat sink and forces meaningful decisions.

### Decay
If upkeep is missing, structures lose integrity over time. Abandoned bases slowly crumble, keeping the world cleaner.

### Upgrade path
- wood
- stone
- metal
- reinforced metal

### Defense systems
- code locks
- clan locks
- shotgun traps or equivalents
- mines
- alarm sensors
- turrets
- power nodes
- generators

## 4. PvP
### Combat styles
- melee
- bows
- crossbows
- improvised firearms
- standard firearms
- trap kills

### Design goals
- combat should be readable in the browser
- time-to-kill should be tense, not instant nonsense
- gear advantage matters but does not delete all skill expression
- leaving base should always involve risk

## 5. Raids
### Raid philosophy
Raids are expensive and dramatic. They are not everyday chores.

### Targets
- doors
- walls
- cupboards/cores
- generators
- turret rooms
- loot rooms

### Raid tools
- satchel-like charges
- shaped charges / C4-like endgame tools
- breaching tools
- burning/incendiary options
- structural pressure tools

### Offline raid mitigation
Recommended mini-app solution:
- vulnerability windows
- reduced raid efficiency outside prime time
- optional notification/defense systems

- moving conflict zone

## 8. Economy
### Economic pillars
- Scrap drives knowledge and research.
- Materials drive base strength and upkeep.
- Components gate power progression.
- Rare items feed late game and raids.

### Trade systems
- NPC traders
- player vending machines
- direct secure trade
- optional black market / auction for selected items

### Anti-inflation sinks
- upkeep
- repair
- research
- fuel consumption
- marketplace taxes
- raid costs
- season resets

## 9. Twitch and WizeBot integration
### Twitch login
Primary identity, profile creation, and community linkage.

### WizeBot rewards
WizeBot-linked rewards should support engagement, not break PvP.
Best uses:
- daily login bonuses
- seasonal leaderboard prizes
- stream event rewards
- cosmetic/status rewards
- limited utility boosts

Avoid:
- giving top-tier weapons directly
- letting external currency bypass all risk and progression

### Stream features
- overlays for rare loot finds
- raid alerts
- world event alerts
- season leaderboard panels
- chat-driven announcements
